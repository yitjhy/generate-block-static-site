
## 方便静态资源发布

- 执行 `server/index.mjs` 根据 `docs/md` 文件生成静态资源
- 代码片段示例代码为 `docs/**/demo/*.md` 文件
- 代码片段说明为 `docs/**/README.md` 文件

## 优点

- 能看到的示例代码永远是可以运行的代码
- 示例代码和运行代码不需要写两次，示例代码即运行代码
- 静态资源网站上对代码块的介绍和git仓库上的README文档一致，代码块说明文档为md文件方便在仓库中查看，不需要写两次
